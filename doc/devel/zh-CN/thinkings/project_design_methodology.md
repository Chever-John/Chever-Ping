这边提供一些角度，期望输出一份完整的文章:)

## 思考角度

从代码结构、代码规范、代码质量、编程哲学、软件设计方法五个方面进行思考。

- 代码结构
  - 目录结构
  - 按功能拆分模块
- 代码规范
  - 编码规范
  - 最佳实践
- 代码质量
  - 编写可测试的代码
  - 高单元测试覆盖率
  - Code Review
- 编程哲学
  - 面向接口编程
  - 面向“对象”编程
- 软件设计方法
  - 设计模式
  - 遵循 SOLID 原则
